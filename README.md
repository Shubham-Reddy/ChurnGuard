## 💡 What It Does

ChurnGuard predicts which customers will leave, explains why using SHAP, calculates revenue at risk, and recommends retention strategies — all in one platform.

---

## ⚡ Modules

**Core Analytics**

- 📊 **Data Overview & EDA** — Interactive exploration with churn pattern visualisations
- 🤖 **ML Engine** — Logistic Regression, Random Forest, XGBoost trained and compared
- 🔍 **SHAP Explainability** — Global and individual customer churn explanations
- 🎯 **Customer Risk Scorer** — Real-time churn probability with risk tier classification
- 🔄 **What-If Simulator** — Live churn probability as attributes change

**Business Intelligence**

- 💰 **CLV & Priority Scoring** — Churn risk × lifetime value prioritisation
- 📉 **Revenue Impact Calculator** — Financial cost of churn and ROI of campaigns
- 🎪 **Retention Campaign Simulator** — Compare strategies by net benefit and ROI
- 🤖 **AI Advisor** — Groq LLaMA 3.3 70B chatbot grounded in your data
- 📄 **PDF Report Generator** — One-click professional executive report
- 📁 **Upload Your Own Data** — Any customer CSV for instant churn analysis

## 📈 Key Results

- **$1.53M** revenue at risk identified
- **428** critical customers flagged for immediate intervention
- **32%** churn reduction possible via contract upgrades
- **293% ROI** achievable through targeted retention campaigns

## 🛠️ Tech Stack

- **Language:** Python 3.11
- **ML:** Scikit-learn, XGBoost, SHAP
- **Frontend:** Streamlit, Plotly
- **AI:** Groq LLaMA 3.3 70B
- **Reports:** ReportLab
- **Data:** IBM Telco Customer Churn Dataset (7,043 customers)
- **Deployment:** Streamlit Cloud
